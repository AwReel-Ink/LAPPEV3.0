🌿 LAPP EV 3.0 — Mode d'emploi

Application de gestion visuelle de chantiers d'espaces verts. Créez des fiches clients avec images satellite annotées, zones d'intervention colorées et liste de matériel.

📐 Mode Création

1. Informations Client

Remplissez les champs dans la barre latérale gauche : nom, téléphone, email, adresse, ville, code d'accès et notes particulières.

2. Ajouter des images satellite

  • Faites vos captures d'écran des vues satellite dont vous avez besoin, puis cliquez sur "+ Ajouter"* pour importer une capture d'écran satellite (Google Maps, Géoportail…). * Action répétable plusieurs fois
  • Naviguez entre les images avec les boutons ◀ ▶ en bas de l'image.
  • Sur chaque image (vue satellite), vous pouvez dessiner des zones d'intervention ; chaque image possède ses propres zones, indépendantes des autres.
  • Cliquez sur "- Supprimer" pour retirer l'image actuellement affichée.

3. Dessiner des zones

  • Sélectionnez un type de zone dans la section "Zones d'intervention".
  • Cliquez sur l'image pour placer des points un par un. Ils seront reliés automatiquement.
  • Pour fermer une zone : double-cliquez ou cliquez sur "Fermer zone".
  • Minimum 3 points pour créer une zone valide.
  • Les zones Parking et Entrée sont des marqueurs fixes (un seul point).

4. Navigation dans l'image

  • Molette souris : zoom avant/arrière.
  • Clic droit + glisser : déplacer l'image.
  • Clic gauche : placer des points.
  • Bouton "Recentrer" : remet l'image à sa position d'origine.

5. Annuler (Undo)

  • Cliquez sur "↩️ Annuler" ou utilisez Ctrl+Z.
  • Historique de 200 actions maximum.

6. Matériel

  • Cliquez sur "🛠️ Matériel" pour ouvrir la liste d'outils.
  • Définissez la quantité de chaque outil nécessaire (tondeuses, débroussailleuses, etc.).

7. Sauvegarder / Charger

  • "💾 Enregistrer" : exporte un fichier .json contenant toutes les données.
  • "📂 Charger .json" : importe un fichier existant pour le modifier.

8. Imprimer

  • Cliquez sur "🖨️ Imprimer" pour générer une version imprimable.
  • Chaque image satellite = une page, avec légende et infos client.
  • Le matériel est sur une page séparée.

📖 Mode Lecture

  • Chargez un fichier .json existant.
  • Consultez les images annotées avec les zones colorées.
  • Naviguez entre les images avec les boutons ◀ ▶.
  • Cochez les cases pour suivre l'avancement du chantier.
  • Accédez à la liste du matériel via le bouton dédié.

🎨 Types de zones & couleurs

  • ■ Tonte & Débroussailleuse — Vert
  • ■ Désherbeur Thermique — Orange
  • ■ Tonte Uniquement — Bleu
  • ■ Débroussailleuse Uniquement — Violet
  • ■ Désherbage Manuel — Rouge
  • ■ Taille Haie — Cyan
  • ■ Parking — Gris
  • ■ Entrée — Jaune
